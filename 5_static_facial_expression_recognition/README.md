# 5. Static Facial Expression Recognition

Section 5 'Static facial expression recognition' builds on the 2024 ACII paper **[EmojiHeroVR: A Study on Facial Expression Recognition under Partial Occlusion from Head-Mounted Displays](https://doi.org/10.1109/ACII63134.2024.00014)** and the 2025 AIxVR paper **[Unimodal and Multimodal Static Facial Expression Recognition for Virtual Reality Users with EmoHeVRDB](https://doi.org/10.1109/AIxVR63409.2025.00048)**.
The corresponding material is maintained in the [EmojiHeroVR Database (EmoHeVRDB) repository](https://github.com/thorbenortmann/emoji-hero-vr-database) and the [EmoHeVRDB SFER repository](https://github.com/thorbenortmann/emohevrdb-sfer).

Accompanying material is available for the following subsections:

- [5.1 Image-Based FER](https://github.com/thorbenortmann/emoji-hero-vr-database/tree/main/vii_baseline/b_training_c_results/emohevrdb)
- [5.2 FEA-Based FER](https://github.com/thorbenortmann/emohevrdb-sfer/tree/main/iii_unimodal_facial_expression_recognition/b_c_training_results)
- [5.3 Multimodal FER](https://github.com/thorbenortmann/emohevrdb-sfer/tree/main/iv_multimodal_facial_expression_recognition/b_training_and_results)
