# 4. EmojiHeroVR Database

Section 4 'EmojiHeroVR Database' builds on the 2024 ACII paper **[EmojiHeroVR: A Study on Facial Expression Recognition under Partial Occlusion from Head-Mounted Displays](https://doi.org/10.1109/ACII63134.2024.00014)**.
The corresponding material is maintained in the [EmojiHeroVR Database (EmoHeVRDB) repository](https://github.com/thorbenortmann/emoji-hero-vr-database).
