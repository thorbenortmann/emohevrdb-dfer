# Cross-Attention Late Fusion

This directory contains the cross-attention late-fusion baseline from Subsection 6.3.3 'Late Fusion':

- [best model](https://drive.google.com/file/d/1VhxxPBqeUpodqPPWmji41vW4g3KQmmZb/view?usp=sharing): 80.56% accuracy (downloadable from Google Drive)
- [late_fusion_cross_attention.ipynb](late_fusion_cross_attention.ipynb): training and evaluation notebook

- [classification_report.txt](classification_report.txt): class-wise test-set precision, recall, and F1-scores
- [confusion_matrix.png](confusion_matrix.png): test-set confusion matrix
- [training_history_accuracy.png](training_history_accuracy.png): plotted training and validation accuracy
- [training_history_loss.png](training_history_loss.png): plotted training and validation loss
