# Average Late Fusion

This directory contains the average late-fusion baseline from Subsection 6.3.3 'Late Fusion':

- [late_fusion_avg.ipynb](late_fusion_avg.ipynb): training and evaluation notebook
- [best model](https://drive.google.com/file/d/1yp4o7Ztz3m683ffQVuDqfrUM_p5ydNdT/view?usp=sharing): 78.44% accuracy (downloadable from Google Drive)

- [classification_report.txt](classification_report.txt): class-wise test-set precision, recall, and F1-scores
- [confusion_matrix.png](confusion_matrix.png): test-set confusion matrix
